== SOS Denúncia

O SOS Denúncia é um projeto que tem o objetivo de facilitar e incentivar a
realização de denúncias de casos de homofobia, violência contra a mulher, racismo, pedofilia, entre outros.
Dando a devida visibilidade e atenção para os casos, podemos minimizar a impudinidade e aumentar a realização de
ações preventivas para diminuir ou até acabar com estas ocorrências.
